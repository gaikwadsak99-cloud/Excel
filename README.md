
# Financial Analytics & Business Intelligence Dashboard

### Project Overview

The Enterprise Financial Analytics & Business Intelligence Dashboard is an interactive Excel-based analytics solution designed to analyze global investment projects and financial performance across multiple countries and sectors. The dashboard converts raw financial data into actionable business insights through dynamic visualizations, KPI reporting, and trend analysis.

This project enables stakeholders to monitor investment portfolios, track project performance, and support strategic decision-making through data-driven insights.

### Objectives
Analyze project funding across countries and sectors.Monitor project status and investment trends.Compare Sovereign and Non-Sovereign projects.Identify top-performing countries and sectors based on funding.Build an interactive dashboard for executive reporting and decision-making.

### Dataset Information

The dataset contains 336 financial project records and includes the following fields:

Project Code,
Project Title,
Country,
Commitment Amount (U.A.),
Starting Date,
Project Status,
Source of Financing,
Sovereign/Non-Sovereign Classification

### Key Performance Indicators (KPIs)

Total Funding (U.A.)
Total Countries,
Sovereign Projects %,
Non-Sovereign Projects %,
Top Funded Country,
Top Funded Sector,
Project Completion Rate
